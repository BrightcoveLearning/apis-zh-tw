介紹

In-Page Experience API 允許您以編程方式管理頁內體驗，使其對 CMS 集成特別有用，允許用戶從他們自己的 CMS 創建管理體驗，而不是轉到 Studio 中的庫。

此快速入門提供了一個示例實現，允許用戶：

  1. 取得帳戶的所有頁面內體驗範本
  2. 從選取的範本建立基本體驗
  3. 選擇要添加到體驗中的播放列表或視頻列表
  4. 更新所選視頻/播放列表的體驗
  5. 發布經驗
