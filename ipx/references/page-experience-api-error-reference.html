頁內體驗API錯誤

頁內體驗API錯誤參考
錯誤 例子 描述
400 BAD_REQUEST 
{
  "status": 400,
  "code": "BAD_REQUEST",
  "message": "No template specified"
}
{
  "status": 400,
  "code": "BAD_REQUEST",
  "message": "Specified template does not exist"
}
{
  "status": 400,
  "code": "BAD_REQUEST",
  "details": [
    "Path `name` is required."
  ]
}
 通常是請求正文的問題，例如省略了template或其他一些必填字段，或指定不存在的模板
401 UNAUTHORIZED 
[
  {
    "error_code": "UNAUTHORIZED",
    "message": "Permission denied."
  }
]
 沒有與請求一起發送訪問令牌，或者用於創建訪問令牌的客戶端憑據對請求沒有正確的權限
404 NOT_FOUND 
{
  "status": 404,
  "code": "NOT_FOUND",
  "message": "Experience {experience_id} not found"
}
 您嘗試更新的體驗不存在。它可能已被刪除，或者請求中的體驗ID可能不正確。
