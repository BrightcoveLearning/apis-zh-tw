樣品上的注意事項

這個示例的代碼全部是 JavaScript，有兩部分（都在 CodePen 中找到：插件腳本在頂部，一小段代碼將插件添加到播放器底部）：

  • 將插件添加到 videojs 播放器的腳本
  • 將數據發送到數據收集器的插件腳本

請注意，將數據“發送"到數據收集API的方式是將請求網址設置為src新圖片的價值（img）元素，然後將該元素注入頁面。

您應該使用相同的方法而不是XMLHttpRequest或其他一些以避免CORS問題。
