介紹

Playback API 有四個操作：

  • 通過ID獲取視頻
  • 通過參考ID獲取視頻
  • 通過ID獲取播放列表
  • 通過參考ID獲取播放列表
Playback API 旨在供網頁或移動應用程序在客戶端使用 - 對於一般媒體管理或 Video Cloud 與您的系統的集成，請使用內容管理系統 API反而。

如果您只需要有關播放器中當前視頻的信息，則可以使用mediainfo對象，而不是進行Playback API調用。

這mediainfo對象包含有關當前加載到播放器中的視頻的信息，而 Playback API 可以檢索有關任何視頻的信息。有關詳細信息，請參見來自mediainfo的視頻元數據文檔。
