Playback API 用於從 Video Cloud 獲取視頻和播放列表數據。它通常用於將視頻數據獲取到網頁或移動應用程序中的播放器。  Playback API是低延遲API，旨在供客戶端從網頁或移動應用程序獲取視頻或播放列表數據時使用。在將Video Cloud與CMS或其他系統集成時，它不是通用的媒體管理API，為此，您應該使用CMS API。

Playback API的身份驗證需要一個策略密鑰，您可以使用政策API或者來自玩家在您的帳戶中。
