主要領域

表格1：主要領域
場地 類型 默認 描述
id 細繩 沒有任何 視頻標識
account_id 細繩 沒有任何 賬號id
created_at ISO 8601日期時間字符串 沒有任何 視頻添加到帳戶的日期時間；例子："2014-12-09T06:07:11.877Z"
cue_points 數組cue_point對象 空數組 插播廣告請求標記或通過播放器API創建的其他操作
custom_fields 物件 空物體 自定義字段名稱的映射：值對；僅包含具有值的字段
description 細繩 空值 視頻的簡短說明 - 最多 250 個單字節字符
duration 數字 空值 視頻長度（以毫秒為單位）
economics 細繩 AD_SUPPORTED 指示是否允許視頻請求廣告。價值觀：AD_SUPPORTED或者FREE
labels 字符串數組 空值 數組標籤分配給視頻
link 鏈接對象 空值 相關鏈接
long_description 細繩 空值 最多5000個單字節字符
name 細繩 沒有任何 視頻標題 - 必填字段（最多 256 個單字節字符）
poster 細繩 空值 海報圖片網址
poster_sources 大批 [] 與視頻海報對應的對像數組。當前只有一張海報，因此只有一個對象。該對象具有單個屬性src其值為海報圖像的 URL。
published_at ISO 8601日期時間字符串 沒有任何 發布視頻的日期時間；例子："2014-12-09T06:07:11.877Z"
reference_id 細繩 空值 帳戶中唯一的任何值（最多150個單字節字符）
sources 大批 空數組 數組視頻源對象
tags 大批 空數組 添加到視頻中的標籤（字符串）數組（每個標籤最多128個單字節字符）
text_tracks 數組text_track對象 空數組 與視頻關聯的WebVTT字幕的數據
thumbnail 細繩 空值 縮略圖 URL
thumbnail_sources 大批 [] 與視頻縮略圖相對應的對像數組。當前只有一個縮略圖，因此只有一個對象。該對象具有單個屬性src其值為縮略圖的 URL。
updated_at ISO 8601日期時間字符串 沒有任何 上次修改視頻的日期時間； [ 1-2] 例子："2015-01-13T17:45:21.977Z"
variants 大批 [] 包含視頻多語言元數據的對像數組

筆記

  • [ 1-2] 修改可能是由用戶或視頻雲系統進行的

text_track 對象

表 5：文本跟踪字段
場地 類型 默認 描述
src 字符串（網址） 空值 .vtt 文件所在的 URL
srclang ISO-639-1語言代碼 空值 字幕語言
label 細繩 空值 要在播放器菜單中使用的標籤 - 它應該與字幕使用相同的語言

筆記

  • [9-1] 設置 mime_type 是可選的，但字幕應該是 UTF-8 編碼的，並且 mime_type 為“text/vtt”

key_systems目的

key_systems 對象包含一個對象列表，每個 DRM 密鑰名稱對應一個對象。如果定義了此對象，則其源受內容保護。

表 4：key_systems 對象
場地 類型 默認 描述
關鍵名 物件 空值 指定 DRM 類型的對象名稱 - 有效鍵包括：

* 公平競爭：com.apple.fps.1_0
* 播放就緒：com.microsoft.playready
* 寬藤：com.widevine.alpha
FairPlay 使用key_request_url和certificate_url
其他人使用license_url		 細繩 空值 您的 DRM 許可證或證書所在的 URL
 

範例 1：

    "key_systems": {
      "com.apple.fps.1_0": {
        "key_request_url": "https://....",
        "certificate_url": "https://..."
      }
    }

範例 2：

    "key_systems": {
      "com.microsoft.playready": {
        "license_url": "https://..."
      },
      "com.widevine.alpha": {
        "license_url": "https://..."
      }
    },

variant目的

每個variant包含特定語言的視頻元數據版本。

表 5：變體對象
場地 類型 默認 描述
language 細繩 沒有任何 形式為 language-COUNTRY 的變體的語言，例如 JA-JA
name 細繩 沒有任何 翻譯的標題
description 細繩 沒有任何 翻譯後的描述
long_description 細繩 沒有任何 翻譯後的 long_description
custom_fields 目的 {} 自定義字段的鍵/值對映射

例子

{
  "language": "ja-JA",
  "name": "バーゼル：ライン川",
  "description": "スイス、バーゼルの中心部にあるライン川のパノラマビュー。",
  "long_description": "スイス、バーゼルの中心部にあるライン川のパノラマビュー。",
  "custom_fields": {
    "language": "Japanese"
  }
}
