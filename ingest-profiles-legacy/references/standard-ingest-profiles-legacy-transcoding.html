使用上傳模塊上傳視頻或使用 DI API 創建動態攝取請求時，您必須提供攝取配置文件，以確定視頻云攝取過程創建的再現的數量和類型。本主題提供有關每個舊版攝取配置文件的詳細信息。有關動態交付配置文件的信息，請參閱用於動態投放的標準提取配置文件。

筆記：當上傳質量較低的源內容並選擇標準配置文件之一時，Video Cloud 將僅生成質量等於或低於源的再現。

也可以創建自己的自定義攝取配置文件。有關信息，請參閱管理攝取配置文件。

文件資料

筆記：
  • 此頁面是使用來自 API 的實時數據構建的；完全加載可能需要一分鐘。
  • 標準配置文件可能不是被編輯 - 如果您想要修改版本，則需要創建自定義配置文件。
