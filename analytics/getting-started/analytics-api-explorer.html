概述

這分析 API允許您檢索 Video Cloud 視頻的各種分析報告。此快速入門將允許您測試多個不同的請求，以查看請求 URL 和響應的外觀。

  • 有關API的完整說明，請參見Analytics API概述和Analytics API參考
  • 如果您想使用自己的帳戶測試請求，請嘗試我們的API測試工具

授權

使用OAuth2訪問令牌授權API調用。令牌與授權標頭中的請求一起傳遞：

    Authorization: Bearer {access_token}

訪問令牌是短暫的（5分鐘），因此每次發出請求時都必須獲得一個

快速啟動時，所有請求都將使用Brightcove示例帳戶，並使用代理人具有必要的客戶端憑據，並將獲取訪問令牌，然後發出請求。
