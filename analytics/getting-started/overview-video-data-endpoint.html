介紹

視頻數據端點提供了一種快速獲取特定視頻的特定數據的方法。當前請求將返回該視頻的全天觀看次數。將來可能會返回其他數據。此端點的主要用例是向高流量頁面上的玩家提供所有時間查看信息，在那裡向分析 API會太慢。

終點

視頻數據請求的端點是：

    https://analytics.api.brightcove.com/v1/alltime/accounts/{account_id}/videos/{video_id}

所有請求均為GET。

驗證

對所有請求分析 API通過 Brightcove 的 OAuth 服務進行身份驗證。

身份驗證基於在Authorization標頭中傳遞的訪問令牌：

    Authorization: Bearer {your token}

您可以獲取客戶端憑據分析 API並使用它們來檢索臨時訪問令牌 - 請參閱OAuth 概述有關可用於獲取客戶端憑據和訪問令牌的不同方法的說明，以及管理 API 憑證有關通過 Studio 獲取客戶憑據的信息

回复

對請求的響應將是以下形式的 JSON：

    {
        "alltime_video_views": 103123
    }

可能的實施

當視頻加載到播放器中時，可以使用以下方法使用視頻數據終結點來顯示視頻的全時視圖：

注意：此端點僅適用於 Professional 和 Enterprise 帳戶。如果您有興趣升級您的帳戶，請聯繫 Brightcove 銷售人員或您的客戶經理。
  1. 設置緩存以按視頻 ID 存儲觀看次數 5-10 分鐘。
  2. 在服務器或頁面上創建邏輯，以首先檢查緩存以查看是否有全時視頻觀看當前視頻的數據，如果有，請使用它。
  3. 如果尚未緩存該值，請請求視頻信息端點以將其檢索，並將該值存儲在緩存中以及在頁面中顯示該值。
  4. 如果對視頻信息端點的請求超時或不成功，請確保設置異常處理-您可能根本不顯示所有時間的視頻觀看次數，或者可能使用擁有的最後一個值。
