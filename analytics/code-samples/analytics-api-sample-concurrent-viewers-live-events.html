介紹

本示例使用獲取實時分析時間序列的終點分析 API根據不同的時間間隔生成實時事件的並發觀眾報告。為方便起見，API返回的紀元時間戳將轉換為人類可讀的時間戳。

局限性

提供此示例和相關代碼作為生產開發的指南。該代碼未經認證可用於生產用途，也未經官方支持。
