局限性

提供此示例和相關代碼作為生產開發的指南。該代碼未經認證可用於生產用途，也未經官方支持。
  • 您必須先檢索播放列表數據並選擇一個播放列表，然後才能將請求提交到Analytics API
  • 對於智能播放列表，將針對當前播放列表中的視頻返回分析數據-對於因停用播放列表的定義而被停用，刪除或刪除而不再位於播放列表中的視頻，則不會返回數據
  • 如果任何視頻也發佈在單個視頻播放器或多個播放列表中，則需要指定播放器以獲取特定播放列表的準確數據
  • 返回的數據包括單個視頻的數據以及播放列表的累積或平均值
  • 如果返回的數據不包括播放列表中所有視頻的項目，則在指定的時間段內未返回這些視頻的數據

應用程式邏輯
