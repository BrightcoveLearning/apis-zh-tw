介紹

這是一個簡單的例子，只需要對分析 API（儘管示例發出兩次請求以獲取 CSV 和 JSON 格式的響應）。目的只是提供一個以不同格式表示響應的示例。

提供此示例和相關代碼作為生產開發的指南。該代碼未經認證可用於生產用途，也未經官方支持。
