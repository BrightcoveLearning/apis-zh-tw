可用字段

可以為維度返回以下字段。

  • viewer - 事件查看器（基於用戶代理和 IP 的 SSO ID 或設備 ID 或唯一字符串）
  • video_view -查看器播放開始的次數
  • time - 以毫秒為單位的視頻視圖的紀元時間
  • engagement_score——參與度

篩選值

用作過濾器時，尺寸可以縮小結果範圍。

過濾器值：以逗號分隔的列表或 viewer.q== 形式的查看器 ID {viewer field}：{value}
