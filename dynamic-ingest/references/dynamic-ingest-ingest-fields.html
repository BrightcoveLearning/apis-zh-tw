動態提取字段

視頻云攝取字段
場地 類型 描述
master 可選的 物件

要攝取的視頻母版
master.url 可選的 網址

視頻源的 URL；需要，除非在數字母版已存檔的情況下重新轉碼，或者您正在向現有視頻添加圖像或文本軌道
master.use_archived_master 可選的 布爾型

對於重新轉碼請求，如果設置為，將使用存檔的主文件true ;如果設置為false，還必須包括url用於源視頻
master.audio_tracks 可選的 物件

一個audio_tracks對象內master對象包含視頻文件中包含的音軌的元數據（如果有的話 - 這也稱為 混入 音頻） - 這將只包含元數據，沒有音頻文件的 URL，因為音軌已經包含在視頻文件中 - 請參閱使用 API 實現多個音軌想要查詢更多的信息
master.audio_tracks.language 可選的 細繩

來自以下子標籤中音頻的多路復用器的語言代碼http://www.iana.org/assignments/language-subtag-registry/language-subtag-registry（可以通過聯繫Brightcove支持為帳戶設置默認值）僅動態投放
master.audio_tracks.variant 可選的 細繩

混入的音軌類型（可以通過聯繫 Brightcove 支持為帳戶設置默認值） - 對於混入音頻，這通常是main僅限動態交付

允許值："main"，"alternate"，"commentary"，"dub"，"descriptive"
audio_tracks 可選的 僅限動態交付 目的[]

音軌對像數組-請參閱使用API實施多個音軌想要查詢更多的信息。
audio_tracks.merge_with_existing 可選的 布爾型

是替換現有音軌還是添加新音軌僅限動態交付

默認值：true
audio_tracks.masters 可選的 目的[]

音軌對像數組僅動態投放
audio_tracks.masters.url 可選的 細繩

音頻文件的 URL 僅限動態交付
audio_tracks.masters.language 可選的 細繩

來自中的子標籤的音軌的語言代碼http://www.iana.org/assignments/language-subtag-registry/language-subtag-registry（可以通過聯繫Brightcove支持為帳戶設置默認值）僅動態投放
audio_tracks.masters.variant 可選的 細繩

音軌的類型（可以通過聯繫 Brightcove 支持為帳戶設置默認值）僅限動態交付

允許值："main"，"alternate"，"commentary"，"dub"，"descriptive"
forensic_watermarking 可選的 布爾型 是否應將取證水印添加到視頻播放中 - 如果您將其設置為真的該帳戶必須啟用取證水印，否則該字段將被忽略 - 請參閱概述：法醫水印更多細節
forensic_watermarking_stub_mode 可選的 布爾型 是否可見的應將取證水印添加到視頻再現中 - 如果您將其設置為 true，則必須為帳戶啟用取證水印，並且forensic_watermarking字段也必須設置為 true - 請參閱概述：法醫水印更多細節可見水印應該只用於測試集成，以確保已成功將取證水印添加到視頻中（使用至少 10 分鐘的視頻）。驗證完成後，必須通過提交新的攝取請求以重新轉碼視頻來刪除它們 - forensic_watermarking_stub_mode必須在重新轉碼請求上設置為 false。
profile 可選的 細繩

提取用於轉碼的配置文件；如果不存在，將使用默認配置文件
text_tracks 可選的 目的[]

的數組text_tracks對象-請參閱攝取WebVTT文件（文本軌道）
text_tracks.url 網址

WebVTT 文件的 URL
text_tracks.srclang 細繩

文本軌道的ISO 639 2字母（alpha-2）語言代碼
text_tracks.kind 可選的 細繩

如何使用vtt文件

默認值：captions

允許值："captions"，"subtitles"，"chapters"，"metadata"
text_tracks.label 可選的 細繩

用戶可讀的標題
text_tracks.default 可選的 布爾型

設置字幕/副標題的默認語言
images 大批{} 描述要攝取圖像的對像數組-請參閱圖像和動態攝取想要查詢更多的信息
variant 細繩 圖像類型 - 以下之一：
  • poster
  • thumbnail
  • portrait
  • square
  • wide
  • ultra-wide
language 細繩 多語言變體的語言代碼-來自的代碼http://www.iana.org/assignments/language-subtag-registry/language-subtag-registry
height 數字 高度（以像素為單位）
Width 數字 寬度（以像素為單位）
url 細繩 發布圖片的 URL
capture-images 可選的 布爾型

轉碼時是否需要截取海報和縮略圖；默認為true如果個人資料有圖像再現，false如果沒有 - 看圖像和動態攝取 API想要查詢更多的信息
poster 可選的 物件

要攝取的視頻海報 - 請參閱圖像和動態攝取 API想要查詢更多的信息
poster.url 網址

視頻海報圖片的 URL
poster.height 可選的 整數

圖像的像素高度
poster.width 可選的 整數

圖像的像素寬度
thumbnail 可選的 物件

要攝取的視頻縮略圖 - 請參閱圖像和動態攝取 API想要查詢更多的信息
thumbnail.url 網址

視頻縮略圖的 URL
thumbnail.height 可選的 整數

圖像的像素高度
thumbnail.width 可選的 整數

圖像的像素寬度
callbacks 可選的 細繩[] URL數組通知應該發送到
