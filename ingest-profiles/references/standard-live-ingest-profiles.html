介紹

使用 Live 模塊創建實時事件時，您必須選擇一個 Live 攝取配置文件，該配置文件決定了 Brightcove Live 創建的再現的數量和質量。本主題提供有關每個標準實時攝取配置文件的詳細信息。

筆記：
  • 對於通過Live API創建的作業，當前無法指定攝取配置文件，而必須定義輸出特徵。
  • 這些配置文件專門用於創建實時事件。將視頻上傳到 Video Cloud 時不使用它們。

也可以創建自己的自定義實時攝取配置文件。有關信息，請參閱創建自定義實時攝取配置文件。

文件資料

筆記：
  • 此頁面是使用來自 API 的實時數據構建的；完全加載可能需要一分鐘。
  • 標準配置文件可能不是被編輯 - 如果你想要一個修改過的版本，你需要創建自定義配置文件 .
