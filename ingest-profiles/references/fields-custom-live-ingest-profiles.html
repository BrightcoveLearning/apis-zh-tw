個人資料欄位

財產 類型 描述 默認
account_id 整數 這視頻雲帳戶ID 沒有任何
name 細繩 個人資料名稱 - 在帳戶中必須是唯一的 沒有任何
display_name 細繩 出現在 Studio Admin 頁面中的配置文件的顯示名稱 - 可能與name，但應該是描述性的，以幫助用戶了解配置文件的預期用途 沒有任何
description 細繩 個人資料說明 沒有任何
model_version 整數 將此設置為1使配置文件顯示在實時模塊中 沒有任何
renditions 大批 再現對像數組 []
packages 大批 DRM軟件包數組-當前不支持DRM，因此該字段應始終為空數組 []

渲染字段

下表顯示了可在請求正文中用於更新實時攝取配置文件的字段。

財產 類型 描述 默認
audio_bitrate 整數 音頻比特率（kbps） 匹配來源
audio_codec 細繩 音頻編解碼器- aac | mp3 由格式或配置文件決定<-在大多數情況下，aac最好/ td>
decoder_bitrate_cap 整數 以kbps為單位，通過緩衝區饋送到解碼器的最大比特率 沒有任何
decoder_buffer_size 整數 以kbps為單位，當使用bitrate_cap 沒有任何
forced_keyframe_rate 數字 每X幀強制設置一個關鍵幀，但仍允許用於場景切換等其他關鍵幀。 沒有任何
format 細繩 視頻格式——ts , mp4 , flv 沒有任何
h264_profile 細繩 H.264 有三個常用的配置文件：Baseline（最低），Main， 和High .[1-5] baseline
hls_optimized_ts 布爾值 針對 HLS 優化的時間段 - 僅適用於 HLS 輸出 沒有任何
keyframe_interval 整數 關鍵幀之間的最大幀數 250
keyframe_rate 漂浮 每秒最大關鍵幀 keyframe_interval共250
label 細繩 圖像再現的圖像類型- poster或者thumbnail 沒有任何
live_stream 布爾值 這是否將是實時視頻流 沒有任何
live_dvr_sliding_window_duration 整數 流的持續時間，可在幾秒鐘內保持可用於LiveDVR交付 300
max_video_bitrate 整數 以kbps為單位，123 = 123kbps-僅適用於h264 沒有任何
max_frame_rate 漂浮 限制frame_rate而不是設置它。除非您的帳戶已通過 Support for 60 FPS 啟用，否則應設置為 30 沒有任何
media_type 細繩 video或者audio 沒有任何
video_bitrate 整數 以kbps為單位，123 = 123kbps 沒有任何
video_codec 細繩 例如h264，ts 由格式或配置文件確定
width 整數 幀寬度（以像素為單位） 沒有任何
height 整數 幀高度（以像素為單位） 沒有任何
