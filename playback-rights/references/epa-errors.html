此功能適用於有權訪問播放權限管理服務的有限可用性階段的一組特定客戶。

錯誤參考

以下狀態列表和錯誤代碼將幫助您在使用 Brightcove 的播放權限管理服務時調試問題：

地位 錯誤代碼 錯誤子碼 描述
400 BAD_REQUEST DUPLICATE_PARAMETERS 請求中多次提供相同的參數名稱。
401 ACCESS_DENIED TOKEN_REQUIRED 一種JSON 網絡令牌 (JWT) 是播放權限所必需的，但未提供。
401 ACCESS_DENIED TOKEN_VALIDATION 這JSON 網絡令牌 (JWT) 提供的沒有通過驗證。
403 FORBIDDEN ACCOUNT_ID 策略密鑰中的帳戶 ID 與 API 請求中的帳戶不匹配。
403 FORBIDDEN CLIENT_GEO 該視頻被限制在當前地理區域播放。該消息將包含有關特定問題的附加信息。
403 FORBIDDEN CLIENT_IP 視頻被限制在當前 IP 地址。
403 FORBIDDEN DOMAIN 該視頻被限制在當前域中播放。
403 FORBIDDEN OFFERS 這JSON 網絡令牌 (JWT) 不允許根據視頻中的可用優惠進行播放。
403 FORBIDDEN TAGS 這JSON 網絡令牌 (JWT) 不允許基於視頻中可用的標籤進行播放。
403 FORBIDDEN VIDEO_NOT_PLAYABLE 對於單個視頻請求，該視頻存在，但現在不允許播放。
404 NOT_FOUND VIDEO_NOT_FOUND 請求的視頻不可用。
404 NOT_FOUND PLAYLIST_NOT_FOUND 請求的播放列表不可用。
405 METHOD_NOT_ALLOWED 此API僅允許GET，HEAD和OPTIONS使用。
500 SERVER_ERROR 內部服務器錯誤。
502 SERVER_ERROR ACCOUNT_RETRIEVE_FAILURE 收到來自後端服務器的錯誤響應。
502 SERVER_ERROR VIDEO_RETRIEVE_FAILURE
502 SERVER_ERROR VIDEO_URLS_RETRIEVE_FAILURE
502 SERVER_ERROR PLAYLIST_RETRIEVE_FAILURE
502 SERVER_ERROR PLAYBACK_RIGHT_RETRIEVE_FAILURE 無法直接從CDN檢索播放。
502 SERVER_ERROR PLAYLIST_VIDEOS_RETRIEVE_FAILURE
502 SERVER_ERROR LICENSE_RETRIEVE_FAILURE 無法從許可證服務器獲得此視頻的許可證。
502 SERVER_ERROR OFFERS_RETRIEVE_FAILURE 無法從API獲得此視頻的報價。
502 SERVER_ERROR RIGHTS_RETRIEVE_FAILURE 無法檢索該視頻的播放權限。
503 SERVICE_UNAVAILABLE 從後端服務器返回此響應。
504 SERVER_TIMEOUT 後端服務器或它們所依賴的服務器之一超時。
