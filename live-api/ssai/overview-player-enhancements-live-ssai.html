介紹

服務器端廣告通過將高質量廣告作為無縫直播流嵌入您的內容，讓您最大限度地提高廣告收入。由於廣告是在服務器端進行拼接的，因此用戶無法利用廣告攔截器，並且一種實現可在多個平台和設備上運行。無需為每個設備創建自定義客戶端。

Brightcove的Live SSAI播放器增強功能建立在服務器端廣告的基礎上，可以擴展用戶的體驗並獲得點擊收入。該工具包允許您向直播流添加客戶端組件，包括廣告計數、廣告倒計時、點擊廣告和隨播廣告。

該工具包還提供了另一種觸發跟踪事件的方法。跟踪數據顯示給客戶端，使您可以靈活地從客戶端或服務器觸發跟踪事件。請注意，廣告攔截器可能會阻止客戶端跟踪。

目前，此功能不支持可跳過廣告。

帳戶設定

請與您的客戶經理聯繫，以為您的帳戶啟用實時流，服務器端廣告插入（SSAI）和針對實時SSAI的播放器增強功能。

規格

Brightcove的Live SSAI播放器增強功能通過提供以下功能來改善最終用戶的體驗：

表面跟踪事件到客戶端

跟踪數據（包括展示次數、四分位數、CreativeView、暫停、靜音）會顯示給客戶端。這使您可以靈活地從客戶端或服務器觸發跟踪事件。應始終從客戶端觸發特定於客戶端的事件，例如單擊，暫停和靜音。

這ExtendedMetadata來自您的 VAST 響應是可自定義的，並且包含在客戶端中。

在視頻流的DVR窗口中投放的廣告休息時間將執行相同的操作。

提供廣告休息時間信息

廣告休息時間有助於保持觀看者對視頻流的關注。

廣告休息信息將傳遞給客戶端。休息期間的每個廣告時長都會向客戶展示。客戶端提供了一個可配置的中斷倒計時或中斷中的廣告數量，以及每個的倒數計時器。

啟用點擊型廣告

通過點擊型廣告，您可以最大限度地提高收入並調整廣告銷售。

廣告服務器的點擊後到達網址將傳遞到播放客戶端。該視頻將是可點擊的，並將在用戶點擊時發送跟踪事件。單擊時，客戶端將暫停流並在新的瀏覽器選項卡或單獨的 web 視圖中打開目標。

由於電視不支持網絡瀏覽器，因此在聯網電視上無法使用點擊廣告。

啟用隨播廣告

隨播廣告提供了另一種用於顯示廣告內容的格式。它們通常是出現在播放器外部的文字、展示廣告或富媒體。這些廣告的好處是它們可以在整個視頻播放過程中保持可見。

隨播廣告不會出現在聯網電視上。

局限性

當前，此功能僅適用於以下地區：

  • ap-southeast-1（新加坡）
  • ap-southeast-2（悉尼）
  • ap-northeast-1（東京）
  • eu-central-1（法蘭克福）
  • us-west-2（俄勒岡）
