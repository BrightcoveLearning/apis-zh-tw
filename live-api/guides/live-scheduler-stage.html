介紹

實時調度程序允許您安排操作靜態入口點 (SEP) 作業。 .您只需根據時間設置調度程序工作流，必要的 API 操作將自動處理。您可以設置兩種調度程序工作流，詳見以下部分。

啟動/停止調度程序

調度程序允許您為冗餘組中的一個 SEP 作業或一組 SEP 作業設置激活和停用時間，您還可以更新時間。詳情請見調度 SEP 流的激活/停用 .

剪輯排程器

您還可以安排從 Live 作業生成剪輯，然後稍後更新安排或在發生變化時取消剪輯。詳情請見為實時流安排剪輯生成 .
