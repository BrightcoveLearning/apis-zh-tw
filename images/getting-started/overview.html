介紹

Brightcove Image API 提供了一種使用簡單的 GET 請求來調整大小、裁剪和/或旋轉圖像的快速簡便的方法。

要求網址

Image API 的請求 URL 是：

https://images.brightcovecdn.com/image/v1/account_id/default/url/{image_url}

所需的路徑參數是：

  • account_id：您的 Video Cloud 帳戶 ID
  • image_url : 可以找到圖像的公共 URL
筆記：
  • 支持以下圖像格式：PNG、JPG、GIF
  • 這image_url必須正確編碼。如果您使用 JavaScript，請使用encodeURIComponent()方法。您還可以使用字符串編碼器由 Brightcove 學習服務提供：

網址參數

要獲取圖像的轉換版本，請在您的請求中使用下面描述的一個或多個 URL 參數。請注意，您可以使用 3 個參數的任意組合。

圖像 API 的 URL 參數
範圍 描述
resize 通過將其設置為等於“widthxheight”來調整圖像大小 - 例如：調整大小=160x90 )
crop 通過將其設置為等於“widthxheight”來裁剪寬度和高度 - 例如：作物=160x90 - 原點是圖像的中心。
rotate 將圖像順時針旋轉指定的度數 - 示例：旋轉 = 90

最佳實踐

如果轉換後的圖像將顯示在網頁或在線應用程序中，請使用 API 檢索圖像並將其存儲在您的服務器上。做不是在運行時檢索它們，因為 API 的速率限制會導致某些請求在高流量頁面或應用程序上失敗，並且還會通過強制 API 請求在每次加載時發出來降低頁面/應用程序的性能一個用戶。
