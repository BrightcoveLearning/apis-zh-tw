介紹

筆記：
  • 策略密鑰僅用於控制存儲在 Video Cloud 中的視頻。如果使用獨立的Brightcove播放器，則無需訪問或檢索策略密鑰。
  • 默認情況下，未為新Brightcove播放器生成的策略密鑰啟用搜索，域限制，地理過濾或要求SSAI。您必須使用 Policy API（或此示例應用程序）手動設置它。有關詳細信息，請參見策略API概述文檔。
提供此示例和相關代碼作為生產開發的指南。該代碼未經認證可用於生產用途，也未經官方支持。

獲取憑證

要使用 Policy API，您需要正確的憑據。在此快速入門中，您將使用Brightcove測試帳戶，因此您無需完成此步驟，但是請注意，使用自己的帳戶時，您將需要獲取客戶端憑據。

  1. 要獲取憑據，請按照以下說明操作管理 API 憑證 .這是獲取客戶端憑據的推薦方式。您需要的權限是玩家閱讀和播放器讀/寫：

    您還可以按照下列指南之一中的說明使用 OAuth API 獲取憑據：

    • OAuth：使用CURL獲取客戶端憑據
    • OAuth：使用郵遞員獲取客戶憑證
