一般錯誤訊息

地位 名稱 信息 原因
400 BAD_REQUEST Invalid JSON body on request 請求正文的 JSON 中有錯誤；可能是基本的 JSON 問題，或者字段值的類型不正確
400 BAD_REQUEST "error": "{fieldname} field is required 請求正文缺少指示的必填字段
400 AD_CONFIG_NOT_FOUND Ad config not found 找不到SSAI請求中指定的廣告配置
400 AD_CONFIG_INACTIVE Ad config is inactive SSAI請求中指定的廣告配置無效
401 UNAUTHORIZED Permission denied. 缺少或無效的OAuth訪問令牌。如果您確實提供了訪問令牌，則該令牌可能沒有此請求的適當範圍
403 NOT_AVAILABLE Resource is not available. 您請求的資源暫時不可用 - 這可能是對視頻進行某種處理時的臨時情況，但如果消息仍然存在，請聯繫支持人員。
404 NOT_FOUND Resource does not exist. 您請求的資源不存在 - 您可能正在嘗試更新廣告配置或在不再存在的會話上獲取廣告跟踪。如果您在端點上使用和 HTTP 方法不允許，也會返回此錯誤。
500 UNKNOWN an unknown internal error occurred Brightcove系統中的問題-請稍後再試。
503 SERVICE_UNAVAILABLE The API is temporarily unavailable 後端問題-請稍後再試。
504 TIMEOUT something took too long 服務器可能太忙 - 稍後再試。
