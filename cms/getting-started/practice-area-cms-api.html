概述

此快速入門將幫助您熟悉可通過內容管理系統 API .這內容管理系統 API啟用了 30 多種不同的操作，因此本教程不會涵蓋所有這些操作，而是允許您測試一些最常用的操作以查看它們返回的數據類型。

提供此示例和相關代碼作為生產開發的指南。該代碼未經認證可用於生產用途，也未經官方支持。
有關本文檔中使用的 API 的詳細信息，您可以查看內容管理系統 API概述文件和內容管理系統 API參考 .

獲取憑證

與 Brightcove 的大多數 RESTful API 一樣，內容管理系統 API需要身份驗證訪問令牌來授權請求。為了獲得訪問令牌，您首先需要獲得具有要請求的各種操作權限的客戶端憑據。此快速入門在 Brightcove 學習服務示例帳戶上發出 API 請求，因此您無需在此處獲取憑據，但您應該了解發生了什麼。這代碼筆將 API 請求信息發送到類似於這個 .代理從私有位置訪問客戶端憑據，並使用它們生成臨時訪問令牌。 OAuth API然後進行API請求並返迴響應數據。
