介紹

可以將自定義字段添加到“媒體"模塊中的視頻中，但是您必須一個接一個地執行此操作。此處的示例應用程序允許您一次向多達 10 個視頻添加自定義字段（並且可以輕鬆修改以增加該數量，甚至將自定義字段添加到帳戶中的所有視頻）。該應用程序相當簡單。為自定義字段創建 UI 的唯一挑戰是它們可以採用簡單的字符串作為值，也可以從值列表中進行選擇。但是，這並不太難管理，因為您可以從字段元數據中輕鬆判斷它是哪種類型的字段並顯示相應的輸入字段。

提供此示例和相關代碼作為生產開發的指南。該代碼未經認證可用於生產用途，也未經官方支持。
