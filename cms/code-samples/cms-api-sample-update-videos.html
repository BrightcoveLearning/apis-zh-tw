介紹

這內容管理系統 API不支持通過單個請求更新多個視頻。但是，您可以以編程方式遍歷一系列視頻數據來執行此操作，並且此示例使用示例Brightcove帳戶對此進行了演示。為了獲取訪問令牌並進行 API 調用，它使用由 Brightcove Learning 服務構建的代理。

由於目的只是為了展示邏輯，我們將只更新視頻的一位元數據：描述。

提供此示例和相關代碼作為生產開發的指南。該代碼未經認證可用於生產用途，也未經官方支持。
