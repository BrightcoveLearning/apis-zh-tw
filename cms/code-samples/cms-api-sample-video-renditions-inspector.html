概述

提供此示例和相關代碼作為生產開發的指南。該代碼未經認證可用於生產用途，也未經官方支持。
請注意，此應用程序旨在幫助 Brightcove 客戶從舊的 Smart Player 遷移到新的 Brightcove Player。它是圍繞使用傳統攝取系統添加的視頻構建的，並且不會為動態交付視頻返回有意義的結果。

此工俱生成的報告包括以下內容：

  • 顯示有多少視頻的摘要不 HLS 或 MP4 演繹或兩者兼而有之
  • 每個視頻的報告，其中顯示了HLS和MP4（HTTP）格式（每個格式的比特率範圍和幀大小）

目的是通過識別需要轉碼以創建 HLS 和/或 MP4 格式的視頻，幫助您準備好從 Smart Player 遷移到 Brightcove Player。

請注意，不檢查動態交付視頻，因為沒有必要。

    您還可以按照下列指南之一中的說明使用 OAuth API 獲取憑據：

  • OAuth：使用郵遞員獲取客戶憑證

渲染檢查器

您可以使用下面的應用程序為您帳戶中的所有視頻生成一份報告。沒有 HLS 或 HTTP MP4（漸進式下載）格式的視頻需要重新轉碼才能在新播放器中正常運行。僅具有低比特率 HLS 再現的視頻也應重新轉碼。
