此應用程序的局限性

提供此示例和相關代碼作為生產開發的指南。該代碼未經認證可用於生產用途，也未經官方支持。
  • 該應用程序使用相同的地理設置更新帳戶中的所有視頻 - 添加選項以通過按文件夾或某些搜索條件請求僅選擇一部分視頻是相當簡單的。
  • 沒有異常處理 - 對於像這裡使用的一個小帳戶，沒有太大關係 - 如果 API 請求超時，應用程序將在那時失敗，您可以簡單地重新運行它；對於大帳戶，您需要添加一些異常處理 - 至少跟踪offset變量，用於跟踪已處理的視頻數量，以便您可以在此時重新啟動應用程序。
  • 對於大型帳戶，這將需要一些時間才能完成處理；應用開始處理後添加的所有視頻都將丟失。但是，該應用程序會告訴您它處理了多少視頻，因此您可以在之後獲得另一個視頻數量，以查看是否有任何視頻丟失。結果按創建日期升序排列，因此錯過的視頻將是最後添加到帳戶的視頻。
