    CMS 應用程式介面影片欄位參考

    這是 CMS API 傳回的視訊欄位的參考資料。

    視訊欄位

    請參閱下列連結，瞭解在建立或更新視訊時可設定哪些欄位：

    • 建立視訊時可設定的欄位
    • 更新視訊時可設定的欄位
    視訊欄位
    欄位 類型 描述
    account_id 字串 視訊雲端帳戶 ID
    ad_keys 字串 代表指派給影片之廣告鍵/值配對的字串。鍵/值對被格式化為鍵 = 值，並由 ＆ 符號分隔。例如："adKeys": "category=sports&live=true"
    clip_source_video_id 字串 為了製作這個視頻，或者null如果這個視頻不是另一個視頻的剪輯來源視頻的 ID。
    complete 布林值 處理是否完成 — 注意：當您建立新視訊時，完整屬性會自動設定為false。一旦視頻存在一個節目，完整的屬性將自動設置為true
    created_at 數字，從紀元的毫秒 影片是在視訊雲端系統中建立的時候
    建立者 物件 影片建立者的相關資訊 (如果有的話)
    提示點 地圖陣列 提示點對應陣列
    custom_fields 字段值對映（字符串） 為視訊設定的自訂欄位值-請注意，沒有此視訊值的自訂欄位不會包含在此對應中；自訂欄位值的長度上限為 1024 個單一位元組字元
    dash_manifest_id 字串 DASH 資訊清單的系統識別碼
    delivery_type 字串 視訊的傳送類型：
    • dynamic_origin -動態交付
    • live_origin -從布萊特灣直播直播
    • remote -資產
    • static_origin -舊式攝入系統
    • unknown -無法確定 (可能表示影片沒有可播放的轉譯)
    description 字串；取代舊的簡短描述 視訊的簡短描述 (最大長度：248 個單位元組字元)
    digital_master_id 字串 數位主機的系統識別碼
    drm_disabled 布林值 對於已啟用 DRM 的帳戶，將此設定為 true 會防止影片遭到 DRM 封裝。如果帳戶未啟用 DRM，則不會顯示此欄位。
    duration 數字，毫秒 視頻的持續時間（以毫秒為單位）
    economics 字符串，必須是有效的枚舉值之一 可以是「AD_ 支持」（默認）或「免費」
    folder_id 字串 影片所屬資料夾的 ID
    地理位址 [1-3] 屬性值對的地圖 視訊的地理限制屬性
    has_digital_master 布林值 視訊是否有可用於重新轉碼的封存主畫面。請注意，如果您使用此欄位來測試視訊是否可以重新轉碼，則對於共用的視訊來說並不可靠。如果影片has_digital_master = true與帳戶分享，您需要測試sharing.by_external_acct = false
    id 字串 視訊識別碼
    影像 屬性值對的地圖 關於海報和縮圖圖像的地圖
    標籤 陣列 分配給視頻的標籤（完整路徑）數組；看到使用標籤欲獲得更多信息。
    連結 屬性值對的地圖 相關連結屬性的地圖
    long_description 字串 完整描述 (最多 5000 個字元)
    name 字串 視訊名稱 (最大長度：248 個單位元組字元)
    offline_enabled 布林值 視訊是否已啟用離線播放
    original_filename 字串 擷取的視訊檔案名稱-遠端資產視訊將為空
    projection 字串 360° 影片的映射投影，例如「等距矩形」
    published_at 字串 ISO-8601 日期-時間字符串，指示視頻發布時間-請注意，這個字段是可寫的; 默認情況下，它被設置為視頻首次激活的日期。
    reference_id 字串 使用者指定的 ID，唯一識別視訊，限制為 150 個字元。參照 ID 可以用作外鍵，以在另一個系統中識別此視頻。參考 ID 不應包含空格、逗號或特殊字元。
    排程 [1-1] 屬性值對的地圖 視訊可用性的開始與結束日期時間地圖
    sharing 屬性值對的地圖 視訊共用屬性的地圖
    smooth_client_manifest_id 字串 平滑資訊清單的系統識別碼
    smooth_server_manifest_id 字串 平滑資訊清單伺服器的系統識別碼
    state[ 1-4] 字串 作用中、非作用中、擱置中、已刪除
    tags[ 1-5] 標籤陣列 (字串) 指派給視訊的標籤陣列
    文字軌 HTML5 風格的文本軌道的數組 指派給視訊的文字軌道 (WebVTT 檔案) 陣列
    updated_at 日期時間字串 上次修改視訊的日期-時間 [1-2]
    更新 (_S) 物件 上次更新影片的人資訊 (如果有的話)

    注意事項

    • [ 1-1] 請注意，排程的影片不會在預定的上市期間以外停用
    • [ 1-2] 可能是由用戶或視頻雲端系統進行的修改
    • [ 1-3] 地理限制是視訊雲端的付費附加元件；如果您想購買地理篩選功能，請聯絡您的客戶經理
    • [ 1-4] state可以設定為ACTIVE或INACTIVE，但不能設定為PENDING或DELETED。
    • [ 1-5] 請注意下列標籤的限制：
      • 標籤最多可包含 128 個字元。如果任何標籤的字元超過 128 個字元，當您嘗試儲存變更時會顯示錯誤
      • 標籤不能包含逗號；例如，不支援roof, attic + front yard的標籤
      • 一個視頻可以有不超過 1200 個標籤
      • 僅在大小寫（小寫，大寫）的標籤作為相同的標籤處理，並且大小寫將自動匹配原始標籤的標籤-區分大小寫的問題需要在外部處理 Video Cloud，例如，使用 JavaScript 方法來修改字符串

    encoding_rate 整數 BPS 編碼速率-僅適用於 MP4 來源

    uploaded_at 字串 以 ISO-8601 日期格式上傳影片的日期
