主要領域

表格1：主要領域
場地 類型 只讀 默認 描述
id 細繩 是的 沒有任何 播放列表 ID
account_id 細繩 是的 沒有任何 賬戶ID
created_at ISO 8601日期時間字符串 是的 沒有任何 日期時間播放列表已添加到帳戶；例子："2014-12-09T06:07:11.877Z"
description 細繩 不 空值 播放列表的簡短描述 - 最多 248 個單字節字符
favorite 布爾值 不 錯誤的 播放列表是否包含在收藏夾中
limit 數字 不 100 播放列表可以容納的視頻數（整數，1 - 100） 僅針對智能播放列表返回
name 細繩 不 沒有任何 播放列表標題-必填字段（最大長度：248個單字節字符）
reference_id 細繩 不 空值 帳戶中唯一的任何值
search 細繩 不 空值 用於定義智能播放列表視頻的搜索字符串（例如：“+tags:”bird”,“birds””）
type 細繩 是的 播放列表的類型 見說明 [ 1] 以下
updated_at ISO 8601日期時間字符串 是的 沒有任何 上次修改日期時間播放列表； [ 1-2] 例子："2015-01-13T17:45:21.977Z"
video_ids 大批 不 播放列表中視頻的ID數組 僅返回手動播放列表

筆記

  • [ 1]

    這type值只能是以下之一：明確ACTIVATED_OLDEST_TO_NEWEST，ACTIVATED_NEWEST_TO_OLDEST，ALPHABETICAL，PLAYS_TOTAL，PLAYS_TRAILING_WEEK，START_DATE_OLDEST_TO_NEWEST，START_DATE_NEWEST_TO_OLDEST

    EXPLICIT表示它是一個手動播放列表；所有其他類型都是智能播放列表，其名稱與它們的排序方案相對應。
