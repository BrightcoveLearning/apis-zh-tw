視頻領域

請參閱以下鏈接以了解在創建或更新視頻時可以設置哪些字段：

  • 創建視頻時可以設置的字段
  • 更新視頻時可以設置的字段
視頻場
場地 類型 描述
account_id 細繩 Video Cloud 帳戶 ID
ad_keys 細繩 表示分配給視頻的廣告鍵/值對的字符串。鍵/值對的格式設置為鍵=值，並用與號分隔。例如："adKeys": "category=sports&live=true"
clip_source_video_id 細繩 為製作此視頻而剪輯的源視頻的 ID 或null如果此視頻不是另一個視頻的剪輯。
complete 布爾型 處理是否完成——注：新建視頻時，complete屬性會自動設置為false .視頻存在一個副本時，complete屬性將自動設置為true
created_at 數，以紀元為單位的毫秒數 在視頻雲系統中創建視頻時
由...製作 物件 有關創建視頻的人的信息（如果有）
cue_points 地圖陣列 提示點圖數組
custom_fields 字段值對（字符串）的映射 為視頻設置了自定義字段值-請注意，不是該視頻的價值未包含在此地圖中；自定義字段值的最大長度為1024個單字節字符
dash_manifest_id 細繩 DASH 清單的系統 ID
delivery_type 細繩 視頻的投放類型：
  • dynamic_origin -動態交付
  • live_origin -來自Brightcove Live的實時流
  • remote - 資產
  • static_origin -舊版攝取系統
  • unknown -無法確定（可能意味著視頻沒有可播放的演繹形式）
description 細繩;取代舊的shortDescription 視頻簡短說明（最大長度：248 個單字節字符）
digital_master_id 細繩 數字母版的系統 ID
drm_disabled 布爾型 對於啟用DRM的帳戶，將其設置為true將防止視頻被DRM打包。如果帳戶未啟用 DRM，則不會出現此字段。
duration 數，毫秒 視頻的持續時間（以毫秒為單位）
economics 字符串，必須是有效的枚舉值之一 “AD_SUPPORTED"（默認）或“FREE"
folder_id 細繩 視頻所屬文件夾的ID
forensic_watermarking 細繩 ACTIVE表示視頻帶有取證水印。UNAVAILBLE表示未添加取證水印。null表示該帳戶未啟用取證水印。
地理 [1-3] 屬性值對的映射 視頻的地理限制屬性
has_digital_master 布爾型 視頻是否具有可用於重新轉碼的存檔母版。請注意，如果您使用此字段測試視頻是否可以重新轉碼，則共享視頻不可靠。如果視頻已共享到該帳戶，則需要同時測試has_digital_master = true和sharing.by_external_acct = false
id 細繩 視頻標識
圖片 屬性值對的映射 有關海報和縮略圖的信息圖
關聯 屬性值對的映射 相關鏈接屬性的地圖
long_description 細繩 詳細說明（最多5000個字符）
name 細繩 視頻名稱（最大長度：248 個單字節字符）
offline_enabled 布爾型 視頻是否開啟離線播放
original_filename 細繩 提取的視頻文件的名稱-對於遠程資產視頻將為空
projection 細繩 360° 視頻的映射投影，例如“equirectangular”
published_at 細繩 ISO-8601日期時間字符串，指示發布視頻的時間-請注意，此字段可寫；默認情況下，它設置為視頻首次激活的日期。
reference_id 細繩 用戶指定的ID 唯一地標識視頻，限制為150個字符。referenceId可用作在另一個系統中標識此視頻的外鍵。參考 ID 不應包含空格、逗號或特殊字符。
日程 [1-1] 屬性值對的映射 視頻可用性的開始和結束日期時間圖
sharing 屬性值對的映射 視頻共享屬性的地圖
smooth_client_manifest_id 細繩 平滑清單的系統 ID
smooth_server_manifest_id 細繩 Smooth 清單服務器的系統 ID
state [1-4] 細繩 活動，不活動，待處理，已刪除
tags [1-5] 標籤數組（字符串） 分配給視頻的標籤數組
text_tracks HTML5樣式的文本軌道的數組 分配給視頻的文本軌道（WebVTT文件）的數組
updated_at 日期時間字符串 上次修改視頻的日期時間 [1-2]
更新者 物件 有關誰最後更新了視頻的信息（如果有）
變體 數組variant元數據對象 不同語言的元數據變體

筆記

  • [ 1-1] 請注意，預定的視頻是不是在預定的可用性期間之外不活動
  • [ 1-2] 修改可能是由用戶或視頻雲系統進行的
  • [1-3] 地理限制是 Video Cloud 的付費插件；如果您想購買地理過濾功能，請聯繫您的客戶經理
  • [1-4] state可以設置為ACTIVE或者INACTIVE， 但不是到PENDING或者DELETED .
  • [1-5] 請注意以下標籤限制：
    • 標籤最多可包含 128 個字符。如果任何標籤的字符數超過128個，則嘗試保存更改時將顯示錯誤消息
    • 標籤不能包含逗號；例如，一個標籤roof, attic + front yard不支持
    • 一個視頻最多只能包含1200個標籤
    • 僅將大小寫不同的標籤（小寫，大寫）作為相同的標籤處理，並且大小寫將自動匹配原始標籤的大小寫-區分大小寫的問題需要在Video Cloud外部處理，例如，使用JavaScript方法進行修改弦

動態投放來源

場地 類型 描述
src 細繩 來源網址
type 細繩 分段視頻類型，通常application/x-mpegURL（對於 HLS）或application/dash+xml（對於 DASH）
ext_x_version 細繩 HLS版本-僅適用於HLS源
profiles 細繩 配置文件-僅適用於DASH源

container 細繩 視頻容器 - 僅適用於 MP4 源
duration 整數 持續時間（以毫秒為單位）-僅適用於MP4來源
encoding_rate 整數 BPS中的編碼率-僅適用於MP4來源
height 整數 幀高（以像素為單位）-僅適用於MP4源
size 整數 文件大小（以字節為單位）-僅適用於MP4源
uploaded_at 細繩 以ISO-8601日期格式上傳視頻的日期
width 整數 幀寬度（以像素為單位）-僅適用於MP4源
