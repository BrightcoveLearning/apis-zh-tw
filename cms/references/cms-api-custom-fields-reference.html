自定義字段

場地 類型 描述
max_custom_fields 數字 發布者可以在帳戶上創建的自定義字段數
custom_fields 大批 帳戶上的自定義字段對象列表
standard_fields 大批 必填或具有描述的字段列表。兩者都沒有的標準字段不會出現在此數組中。

自定義領域圖

對於每個自定義字段：

場地 類型 描述
id 細繩 字段的內部名稱
description 細繩  
display_name 細繩  
type 細繩 '枚舉之一' 或 '字符串' （這翻譯為“列表' 或 '文本' 在用戶界面中）
enum_values 大批 對自定義字段有效的字符串值數組；僅在type ='enum'時出現
required 布爾型 此字段是否必須有值才能激活視頻
