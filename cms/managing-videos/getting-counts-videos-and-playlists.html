介紹

有時，獲取帳戶中的視頻或播放列表數量或在過去一個月或一年中添加了多少個視頻有時很有用。您可以使用/counts/ CMS API 的端點。

驗證

向內容管理系統 API需要包含您的訪問令牌的授權標頭。有關如何獲取客戶端憑據並使用它們來檢索訪問令牌的詳細信息，請參閱Brightcove OAuth 概述 .

獲取視頻計數

要計算視頻數量，請製作GET請求：

    https://cms.api.brightcove.com/v1/accounts/{account_id}/counts/videos

響應將如下所示：

    {
      "count": 4999
    }
請注意，已刪除的視頻不包括在計數中，也不包括在任何內容管理系統 API回复。

使視頻計數端點如此強大的原因是您還附加了一個搜索參數到請求。這允許您計算搜索將返回的任何視頻。由於search參數非常靈活，因此您可以計算很多東西，例如：

  • 具有特定標籤或自定義字段特定值的視頻數量
  • 去年（或月、周等）添加的視頻數量
  • 當前可播放的視頻數量
  • 計劃下週發布的視頻數量

獲取播放列表數

您還可以通過創建一個來獲取帳戶中播放列表的數量GET請求：

    https://cms.api.brightcove.com/v1/accounts/{account_id}/counts/playlists

響應的格式與視頻計數的格式相同。

您還可以在播放列表計數請求中使用搜索參數。因此，例如，您可以通過附加q=type:EXPLICIT到請求。

此外，通過將GET請求發送至以下位置，您可以獲取任何手動或智能播放列表中的視頻計數：

    https://cms.api.brightcove.com/v1/accounts/{account_id}/counts/playlists/playlist_id/videos
