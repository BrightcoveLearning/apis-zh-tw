介紹

播放列表使您可以將一組相關的視頻分組在一起，以便可以將它們加載到播放器中並作為一組觀看。這內容管理系統 API包含一組可用於創建、更新或刪除播放列表的寫入方法：

  • 建立播放清單
  • 更新播放列表
  • 刪除播放列表
這些任務也可以在 Studio Media 模塊中執行。

驗證

向內容管理系統 API需要包含您的訪問令牌的授權標頭。有關如何獲取客戶端憑據並使用它們來檢索訪問令牌的詳細信息，請參閱Brightcove OAuth 概述 .

播放清單類型

有一個EXPLICIT（手動）播放列表類型，您指定要作為視頻 ID 數組包含的視頻。還有七種智能播放列表類型 - 智能播放列表類型的不同之處在於它們如何對使用搜索字符串動態選擇的一組視頻進行排序。下表顯示了所有播放列表類型。

播放清單類型
類型 描述
EXPLICIT 手動播放列表。包含的視頻及其出現的順序由視頻 ID 數組定義。
ACTIVATED_OLDEST_TO_NEWEST 一個智能播放列表，其中按激活日期（升序）對所選視頻進行排序。
ACTIVATED_NEWEST_TO_OLDEST 一個智能播放列表，其中所選視頻按激活日期（降序）排序。
ALPHABETICAL 一個智能播放列表，其中按名稱的字母順序顯示所選視頻。
PLAYS_TOTAL 一個智能播放列表，其中，所選視頻按總播放次數（降序）排序。
PLAYS_TRAILING_WEEK 智能播放列表，其中所選視頻按上週的播放次數（降序）排序。
START_DATE_OLDEST_TO_NEWEST 一個智能播放列表，其中按預定的開始日期（升序）對所選視頻進行排序。
START_DATE_NEWEST_TO_OLDEST 一個智能播放列表，其中選定的視頻按排定的開始日期（降序）排列。

搜索欄

對於所有智能播放列表類型，視頻集合會根據search播放列表的字段值。搜索字段值必須是標籤的有效搜索字符串搜索字符串用於 CMS API。以下是一些示例，說明了它們將返回的視頻。

目前播放列表只能通過搜索一個或多個標籤（用逗號分隔）來構建。此外，您無法搜索這樣的視頻不是有一個特定的標籤。
示例搜索字符串
搜索字符串 描述
+tags:bird 返回帶有“bird”標籤的視頻
+tags:bird,woodland 返回同時具有“bird”和“woodland”標籤的視頻
tags:bird,woodland 返回具有任何一個標籤“鳥”或“林地”（注意：離開+退出搜索字符串會有所不同）

獲取視頻數量

您可以使用counts端點：

    https://cms.api.brightcove.com/v1/accounts/account_id/playlists/playlist_id/videos
對於智能播放列表，返回的結果將指示為播放列表指定的搜索條件當前返回的視頻數量。

建立播放清單

要創建新的播放列表，您需要創建一個POST請求：

    https://cms.api.brightcove.com/v1/accounts/{account_id}/playlists

要求正文

您可以在請求中包含許多（但不是全部！）視頻元數據字段。你必須至少包括一個name和type對於播放列表。以下是兩個示例，一個示例創建一個EXPLICIT播放列表，第二個示例創建一個ACTIVATED_NEWEST_TO_OLDEST類型的智能播放列表，其中包括帶有兩個標籤的視頻：

對於EXPLICIT播放列表

    {
      "type": "EXPLICIT",
      "name": "My manual playlist",
      "video_ids": [
        "5289680419001",
        "5289693763001",
        "5289680417001",
        "5288472314001"
      ]
    }

對於智能播放列表

    {
      "type": "ACTIVATED_NEWEST_TO_OLDEST",
      "name": "My smart playlist",
      "search": "+tags:bird-tags:sea"
    }
請注意，所有播放列表都有 100 個視頻的限制，但您可以使用limit場地。

見API 參考適用於可用於創建播放列表的所有字段。

更新播放列表

更新播放列表與創建播放列表非常相似。唯一的區別是請求方法（PATCH而不是 POST) 和附加到 URL 的播放列表 ID。請求正文的字段完全相同。

請注意，您不能改變一個EXPLICIT播放列表更改為另一種類型，或將另一種播放列表更改為EXPLICIT .

要求網址

    https://cms.api.brightcove.com/v1/accounts/account_id/playlists/playlist_id

在中找到完整的詳細信息API參考。

刪除播放列表

要刪除播放列表，請使用DELETE方法：

    https://cms.api.brightcove.com/v1/accounts/account_id/playlists/playlist_id

（這是用於更新播放列表的URL。）

注意刪除不能被撤消 - 在發送該請求之前三思而後行！刪除播放列表確實不是影響播放列表中包含的視頻。
