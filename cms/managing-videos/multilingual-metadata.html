介紹

如果您有國際觀眾（或在同一國家/地區使用多種語言），則可能需要提供多種語言的視頻元數據，例如標題和說明。

Video Cloud 允許以下內容的多語言版本：

  • 圖片
  • 文本軌道
  • 名稱（通過variants）
  • 描述variants）
  • 詳細描述variants）
  • 自定義字段值variants）

創建一個變體

使用內容管理系統 API ( 創建視頻變體 ) 您可以添加和管理一組variants提供替代方案name , description , long_description，以及特定於語言的自定義字段/值的映射。

要創建變體，請發送POST對端點的請求：

https://cms.api.brightcove.com/v1/accounts/{account_id}/videos/{video_id}/variants/
{
  "language": "ja-JA",
  "name": "バーゼル：ライン川",
  "description": "スイス、バーゼルの中心部にあるライン川のパノラマビュー。",
  "long_description": "スイス、バーゼルの中心部にあるライン川のパノラマビュー。",
  "custom_fields": {
    "language": "Japanese"
  }
}

樣本響應

 
{
  "language": "ja-JA",
  "name": "バーゼル：ライン川",
  "description": "スイス、バーゼルの中心部にあるライン川のパノラマビュー。",
  "long_description": "スイス、バーゼルの中心部にあるライン川のパノラマビュー。",
  "custom_fields": {
    "language": "Japanese"
  }
}

https://cms.api.brightcove.com/v1/accounts/{account_id}/videos/{video_id}/variants

獲取特定的變體

要獲得特定變體，請發送GET對端點的請求：

https://cms.api.brightcove.com/v1/accounts/{account_id}/videos/{video_id}/variants/{language}

這裡language是變體中使用的語言代碼，例如 去DE.

樣本響應

 
{
  "language": "de-DE",
  "name": "Basel: Der Rhein",
  "description": "Panoramablick auf den Rhein im Zentrum von Basel, Schweiz.",
  "long_description": "Panoramablick auf den Rhein im Zentrum von Basel, Schweiz.",
  "custom_fields": {
  }
}

更新變體

您可以通過發送PATCH對端點的請求：

https://cms.api.brightcove.com/v1/accounts/{account_id}/videos/{video_id}/variants/{language}

這裡language是變體中使用的語言代碼，例如 去DE.

您只需要包含正在更改的字段。請求正文字段與創建變體的相同，除了你可以 不是 包括language場地 .（如果您添加了language field一種VALIDATION_ERROR將返回。）

示例請求正文

 
{
  "name": "Basilea: el Rin",
  "description": "Vista panorámica del Rin en el centro de Basilea, Suiza.",
  "long_description": "Vista panorámica del Rin en el centro de Basilea, Suiza.",
  "custom_fields": {
    "language": "Spanish"
 }
}

樣本響應

 
{
  "language": "es-ES",
  "name": "Basilea: el Rin",
  "description": "Vista panorámica del Rin en el centro de Basilea, Suiza.",
  "long_description": "Vista panorámica del Rin en el centro de Basilea, Suiza.",
  "custom_fields": {
    "language": "Spanish"
  }
}

刪除變體

要刪除變體，請發送DELETE對端點的請求：

https://cms.api.brightcove.com/v1/accounts/{account_id}/videos/{video_id}/variants/{language}

不包括請求正文。成功刪除後，將返回一個204 No Content回复。
