介紹

可以創建文件夾來組織視頻庫中的視頻。您可以選擇按主題、作者、產品、地區等組織視頻。將視頻保存在文件夾中可以使他們更容易在擁有數百或數千個視頻的帳戶中進行管理。

筆記：
  • 本主題中討論的所有任務也可以在以下位置執行工作室而不是通過CMS API。
  • 這內容管理系統 API所有請求都需要客戶端憑據和訪問令牌。如果您需要獲取憑據的幫助，請參閱獲取下面的憑據部分。
  • 本主題涵蓋了 CMS API 最常用的文件夾操作，但您還可以在CMS API 參考 .

獲取CMS API的憑據

要使用內容管理系統 API您將需要適當的憑據。

在大多數情況下，獲取憑據的最簡單方法是通過 Studio Admin API 身份驗證部分（需要您帳戶的管理員權限）。看管理 API 身份驗證憑據詳情。在這種情況下，您需要的權限是影片 -您同時需要讀寫權限：

如果Studio中沒有所需的權限，或者您希望直接從OAuth API獲取權限，請選擇獲取客戶憑證下面列出的文件。無論您選擇哪個選項，您都需要請求正確的操作權限。以下可以與 cURL 或 Postman 一起使用以獲得適當的權限：

           "operations": [
           "video-cloud/videos/read",
           "video-cloud/videos/create",
           "video-cloud/videos/update"]
  • OAuth：使用cURL獲取客戶端憑據
  • OAuth：使用郵遞員獲取客戶憑證

建立資料夾

這是創建新文件夾所需的信息。

要求網址

做一個POST請求：

      https://cms.api.brightcove.com/v1/accounts/account_id/folders

要求正文

      {
        "name": folder_name
      }

回复

        "name": "places",

      }

將視頻添加到文件夾

要將視頻添加到文件夾，請製作PUT請求：

      https://cms.api.brightcove.com/v1/accounts/account_id/folders/folder_id/videos/video_id

要求正文

此操作沒有請求正文。

回复

對此請求的響應將是204 NO CONTENT .

從文件夾中刪除視頻

要從文件夾中刪除視頻，請製作DELETE請求與您用於添加視頻的 URL 相同的 URL：

      https://cms.api.brightcove.com/v1/accounts/account_id/folders/folder_id/videos/video_id

要求正文

對於此請求，您還必須在請求正文中包括文件夾名稱：

      {
        "name": folder_name
      }

獲取文件夾中的視頻

要獲取文件夾中的視頻，請製作GET請求：

      https://cms.api.brightcove.com/v1/accounts/:account_id/folders/folder_id/videos

請求參數

限制和抵消

默認情況下，此請求將返回最近添加到該文件夾的20個視頻。您可以更改此設置，並通過將以下一個或兩個參數添加到請求中來瀏覽包含許多視頻的文件夾的結果：

  • limit：要返回的視頻數（默認：20；最大：100）
  • offset：要跳過的視頻數（默認值：0）

例如，此請求將返回可能結果的視頻21-30（假設文件夾包含那麼多視頻）：

https://cms.api.brightcove.com/v1/accounts/:account_id/folders/folder_id/videos?limit=10&offset=20

種類

您還可以使用種類參數，將其設置為等於以下任何字段：

  • 名稱
  • reference_id
  • 更新時間
  • created_at
  • Published_at
  • schedule_starts_at
  • schedule_ends_at
  • 狀態
  • plays_total
  • play_trailing_week

要按降序排序，請在字段值前加上減號 ( -created_at）。

默認值為種類是-updated_at .
