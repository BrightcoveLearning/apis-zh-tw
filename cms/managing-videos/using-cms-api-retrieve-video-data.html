介紹

在本主題中，您將學習如何檢索有關您的 Video Cloud 視頻的各種信息，使用內容管理系統 API要求。要創建這些請求，請將本文檔中列出的端點附加到具有您的帳戶 ID 的基本 URL。

    https://cms.api.brightcove.com/v1/accounts/{account_id}/{endpoint}

向內容管理系統 API需要包含您的訪問令牌的授權標頭。有關如何獲取客戶端憑據並使用它們來檢索訪問令牌的詳細信息，請參閱Brightcove OAuth 概述 .
