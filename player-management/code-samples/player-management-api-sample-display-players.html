概述

提供此示例和相關代碼作為生產開發的指南。該代碼未經認證可用於生產用途，也未經官方支持。

要了解有關此示例應用程序架構的一般方法，特別是身份驗證和 AJAX 實現，請參閱使用 HTTPRequest 的示例 .

應用架構

此應用程序的高級應用邏輯如下：

  1. 收集訪問帳戶所需的用戶數據
  2. 單擊按鈕後，執行AJAX調用以檢索所有播放器信息
  3. 在成功進行AJAX調用時，請執行以下操作：
    • 將JSON響應數據注入HTML頁面
    • 動態構建HTML，以按順序顯示播放器信息並將其註入HTML頁面
