免責聲明

此示例應用程序旨在讓您探索 Player Management API 的基礎知識。它並不是要成為可用於企業的，可部署的應用程序。例如，表單驗證不是詳盡無遺的，並且密碼不是使用HTTPS發送的。

要了解有關此示例應用程序架構的一般方法，特別是身份驗證和 AJAX 實現，請參閱使用 HTTPRequest 的示例 .

應用架構

此應用程序的高級應用邏輯如下：

  1. 收集創建播放器所需的用戶數據
  2. 構建JSON數據結構以傳遞到API來創建播放器
  3. 單擊按鈕後，執行AJAX調用以創建播放器
  4. 在成功進行AJAX調用時，請執行以下操作：
    • 將JSON響應數據注入HTML頁面
    • 調用功能以發布播放器
  5. 執行AJAX通話以發布播放器
  6. 在成功進行AJAX調用時，請執行以下操作：
    • 將JSON響應數據注入HTML
    • 動態構建顯示所選播放器信息的HTML並將其註入HTML頁面
