介紹

取證水印 (FW) 可讓您保護優質內容免遭盜版和未經授權的內容共享。此功能為您的視頻添加了隱形水印，讓您可以跟踪任何內容洩漏。

要了解有關此功能的更多信息，請參閱概述：法醫水印文檔。

將軍

內容檢測如何工作？

檢測過程基於能夠下載、捕獲或記錄到文件中的部分內容。NAGRA 的一個特定工具，稱為打包機 , 生成一個.pak文件。該文件被提交給 NAGRA 以檢索交付的唯一序列，然後將其與用戶匹配。您負責將序列與用戶匹配。檢測服務直接通過NAGRA獲得；通常由內容所有者僱用的第三者爬行互聯網以查找非法內容。


集成

Chromecast 或 AirPlay 等設備是否支持取證水印？

是的，水印令牌會傳遞給投射接收器。網絡播放器和本機 SDK 支持此功能。


跟踪和報告

NAGRA 如何跟踪客戶使用的交易數量？

NAGRA 無法跟踪我們軟件的任何部分是否連接到 NAGRA 以報告交易。

您有責任每 24 小時從 Analytics API 下載一份報告，其中包含每天唯一身份查看者的數量，以提供所用交易數量的證明。


舉報是如何處理的？

可通過 Analytics API 獲得有關取證水印視頻觀看者的分析報告。

有關詳細信息，請參見概述：法醫水印文檔。

轉碼

成功使用固件所需的最短視頻時長是多少？

5分鐘


網絡播放器

是否有疊加顯示內容具有不可見水印？

Brightcove Player 支持疊加。
